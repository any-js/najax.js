Class: Tx

Tx

new Tx(opt:)

Tx class. This class is created automatically by $najax.scriptTag and $najax.jsonpTag.

Method Default ver Tiny ver
All methods Yes -

Specification

  • Specify load and response behaviors by Nx object.
  • Specify by Tx object's various method or Tx opt method.
  • Begin tag-loading by Tx done.
  • Tx done returns Relay object.
  • Relay provide operating response-data, and bundling multiple Relay.
  • Tx implement tag-loading by appending SCRIPT tag element.
